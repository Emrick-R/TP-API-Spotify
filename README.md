# TP-API-Spotify

L’objectif de ce travail pratique est de mettre vos connaissances en matière d'API en pratique en exploitant dans un premier temps une API . Mais également de comprendre comment fonctionne d’une API sécurisée dans la mesure où vous allez devoir utiliser une API sécurisée par le biais de l’OAuth2.0.

# Sujet 1 : Utilisation de l'API Spotify

Dans le répertoire `requêtes`, vous trouverez un fichier json qui regroupe différentes requêtes effectuées dans un logiciel de type Postman ou Insomnia.

# Sujet 2 : Création d'une application web en Go utilisant l'API Spotify
Dans le répertoire `site web`, vous trouverez une application web développée en Go qui interagit avec l'API Spotify pour afficher des informations sur des artistes, albums et titres.

## Lancement de l'application

Cloner le dépôt :

```bash
git clone https://github.com/votre-utilisateur/TP-API-Spotify.git
cd '.\site web\'
```
Dans le répertoire `site web`, exécutez l'application Go dans le terminal :

```bash
go run main.go
```
>L'application sera accessible à l'adresse suivante : `http://localhost:8080`
